# inpainting_cGAN
This repository is a Tensorflow implementation of "Conditional Attribute를 통한 Inpainting GAN 모델의 성능 개선"

## Requirements
- tensorflow 1.9.0
- python 3.5.3
- numpy 1.14.2
- pillow 5.0.0
- scipy 0.19.0
- opencv 3.2.0
- pyamg 3.3.2
- opencv 4.1.0
